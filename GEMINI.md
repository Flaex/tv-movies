# Media Tracker Project

## Project Goal
The primary objective of this project is to create a functional and well-structured Markdown-based media tracker (`media-tracker.md`) to help the user organize and monitor their collection of movies and TV shows, distinguishing between items already watched and those that are pending.

## Instructions for Gemini
- **Maintain Structure**: When adding new items, ensure they follow the established table format.
- **Data Integrity**: When populating information, prioritize accuracy for fields like Type, Year, Genre, and Streaming Platform.
- **Formatting**: Keep table cells concise to avoid excessive text wrapping in Markdown viewers.
- **Ordering**: Keep the table organized by 'Type' (Movies first, then TV Shows) as a default, unless otherwise requested.
